# axcl-python-application-examples
A collection of python application demos and projects using the M5Stack LLM 8850 accelerator (Axera ax650) and rpi5. 

Pre-requisite system packages: 
> numpy>=2.3.4  
> opencv_python>=4.12.0.88  
> picamera2>=0.3.31
